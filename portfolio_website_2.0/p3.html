Search

During the search project I collaborated with other students to develope an algorithm similar to google's pagerank algorithm which maximized search time by querying in wikipedias of various sizes and using a regex to stem all relevant words, then storing their relevance in hastables by document.

The function of the Querier greatly reduced search time as the user only had to search through a pre-computed hashtable. We utilized google's page rank algorithm to determine the relevance of the article and developed our own search engine. This program was developed in python on the visual studio code platform.
