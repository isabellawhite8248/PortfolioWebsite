Travel Planner

The travel planner project was a partner project in my CS 200 class regarding data structures and algorithms. We implemented Dijkstra's algorithm and breath width first search to allow the user to upload any travel graph to the program and it would inform the user the cheapest or shortest route towards his or her destination. The algorithms written maximized the efficiency of the seach and took into account edge cases such as circular paths and/or paths to nonexistent nodes. The project was coded entirely in java and we utilized IntelliJ.
